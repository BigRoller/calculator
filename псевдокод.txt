Начало
Определить и подключить необходимые библиотеки и файлы заголовков.
Установить кодировку консоли в Windows-1251 для правильного отображения символов.
Вывести пользовательское меню с доступными функциями и запросить выбор пользователя.
Считать выбор пользователя.
В зависимости от выбора пользователя выполнить соответствующую функцию:
Начало блока
   - Если выбрана функция "Калькулятор":
     - Вызвать функцию "calculate" для запуска калькулятора.
   - Если выбрана функция "Посмотреть расписание":
     - Вызвать функцию "seeSchedule" для отображения расписания.
   - Если выбрана функция "Преобразование двухзначных чисел в слова":
     - Запросить пользователя ввести двухзначное число.
     - Проверить, что введенное число корректное.
     - Если число корректное, вызвать функцию "writeNumToWords" для преобразования числа в слова и вывести результат.
     - Если число некорректное, вывести сообщение об ошибке.
   - Если выбрана функция "Перевод введенного символа в код Windows-1251":
     - Вызвать функцию "convertSymbol" для выполнения перевода.
   - Если выбрана функция "Перевод чисел из 10 в 2 систему счисления":
     - Запросить пользователя ввести число в десятичной системе счисления.
     - Вызвать функцию "translateDecimalToBinary" для перевода числа в двоичную систему счисления и вывести результат.
   - Если выбрана функция "Выход":
     - Завершить программу.
   - Если выбор пользователя неверный:
     - Вывести сообщение об ошибке.
Конец блока
Проверить, что при вводе неправильного значения был произведен сброс состояния cin и очистка буфера ввода.
Повторить шаги 3-6, пока пользователь не выберет функцию "Выход".
Конец

Начало модуля calculate
Запросить у пользователя ввод выражения в формате "a op b"
Считать значения a, op и b с помощью cin
В зависимости от значения оператора op выполнить соответствующую операцию:
   - Если op = '+', выполнить сложение a и b и вывести результат
   - Если op = '-', выполнить вычитание b из a и вывести результат
   - Если op = '*', выполнить умножение a на b и вывести результат
   - Если op = '/', проверить, является ли b нулем:
     - Если да, вывести сообщение об ошибке деления на ноль
     - Если нет, выполнить деление a на b и вывести результат
   - Если op не равно ни одному из допустимых операторов, вывести сообщение об ошибке некорректного ввода
Завершить выполнение программы
Конец модуля calculate

Начало модуля translate_decimal_to_binary
 Прочитать число num, которое нужно перевести в двоичную систему.
 Создать пустую строку result для хранения результата.
 Пока число num больше нуля выполнить следующие действия:
    Получить остаток от деления числа num на 2 и преобразовать его в символ добавив к нему код символа '0'.
    Добавить полученный символ в начало строки result.
    Разделить число num на 2 без остатка.
 Вернуть результат - строку result, содержащую двоичное представление числа num.
Конец модуля translate_decimal_to_binary

Начало модуля writeNumToWords(num):
    Присвоить result = ""

    Если num < 10 or num > 99:
       Возвратить result

    Задать массив ones = ["", "один", "два", "три", "четыре", "пять", "шесть", "семь", "восемь", "девять"]
    Задать массив teens = ["десять", "одиннадцать", "двенадцать", "тринадцать", "четырнадцать", "пятнадцать", "шестнадцать", "семнадцать", "восемнадцать", "девятнадцать"]
    Задать массив tens = ["", "", "двадцать", "тридцать", "сорок", "пятьдесят", "шестьдесят", "семьдесят", "восемьдесят", "девяносто"]

    Если num >= 10 and num <= 19:
        Присвоить result = teens[num - 10]
    Иначе:
        Присвоить result = tens[num / 10] + " " + ones[num % 10]

    Возвратить result
Конец модуля writeNumToWords(num)
